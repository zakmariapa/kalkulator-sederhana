# KALKULAOR
Project ini adalah kalkulator sederhana berbasis web yang dibuat menggunakan HTML,CSS, dan Java script. kalkulator ini dapat digunakan untuk melakukan perhitungan dasar seperti
penjumlahan,pengurangan,perkalian ,dan pembagian,serta dilengkapi dengan fitur matematika lanjutan seperti sin,cos,tan,log,in,akar,pangkat,dan konstanta n serta e.
tampilan kalkulator dibuat sederhana dan rapi sehingga mudah digunakan oleh pengguna, baik untuk pelajar maupun kebutuhan perhitungan sehari hari. sehingga meningkatkan 
efesiensi dan kenyamanan pengguna.
